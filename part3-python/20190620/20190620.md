- numpy 함수

  - kron :

    ```
    행렬 각 내부 요소들을 곱해줌 (확장)
    # ex : 1x3 , 1x3을 kron해주면 1x9가 나옴
    >>> np.kron([1,10,100], [5,6,7])
    array([  5,   6,   7,  50,  60,  70, 500, 600, 700])
    >>> np.kron([5,6,7], [1,10,100])
    array([  5,  50, 500,   6,  60, 600,   7,  70, 700])
    ```
